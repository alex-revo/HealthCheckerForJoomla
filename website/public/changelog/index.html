3.3.x Releases

v3.3.5 February 26, 2026
  • Fix Menu Orphans check now lists the specific menu item titles that are broken, making it easier to find and fix orphaned links (#45)
  • Fix Unused Modules check now mentions {loadmodule} usage, so modules loaded via content plugins are accounted for (#47)
v3.3.4 February 25, 2026
  • Fix Fixed false positive orphaned tables for extensions using non-standard SQL install file paths by parsing XML manifests to discover actual SQL locations (#41)
  • Fix Fixed error when extension manifest lacks an install sql section
v3.3.3 February 24, 2026
  • Feature Module Position check now lists orphaned modules in the warning description (#40)
v3.3.2 February 22, 2026
  • Feature Added HealthStatus parameter to getDocsUrl() method, enabling status-aware documentation links (#38)
  • Fix Better error handling for ajax errors
  • Breaking Third-party plugins overriding getDocsUrl() must update their method signature
v3.3.1 February 22, 2026
  • Fix HTML markup now renders correctly in HTML export reports instead of displaying raw tags (#36)
  • Feature HTML export now includes a link to the documentation (#37)
  • Fix Improved HTML export layout and formatting (#37)
v3.3.0 February 17, 2026
  • Feature Added Core Directories health check - verifies essential Joomla directories exist and are accessible (#31)
  • Feature Added Index File health check - ensures index.html security files are present in key directories (#32)
  • Internal Standardized die() statements to die; for consistency (#30)
  • Internal Removed duplicate docblocks from performCheck() methods (#29)
